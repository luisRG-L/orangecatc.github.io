¿Qué es un Orange ID?

Un Orange ID es un código de servidor o clave para usar para los estilos, conexiones dependiendo de tu gusto

Crear mi propio ID

Para crear tu propio Orange Id, clica en el siguiente botón y rellena el formulario que aparecerá en tu pantalla. Una vez hecho haz clic en 'Crear'.

Una vez hecho eso, te dirá tu ID. Cópialo para usarlo en tus proyectos Orange Cat. También se descargará un archivo .oid, Este lo debes incluir en la carpeta oids de tu proyecto.
