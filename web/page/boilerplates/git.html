¿Qué es git?

Git es la plataforma más popular de control de versiones. Esta se creó para hacer más compatible el mantenimiento de versiones de aplicaciones.

El creador de Git fue Linus Torvalds, también creador del famoso sistema operativo Linux

Después de eso, también se han creado otros sistemas basados en git para mejorarlo como GitHub, que añade una GUI mejor para crear proyectos.

Una de las ventajas más favorables de git es su organización por ramas, ya que cada desarrollador puede tener su rama para que unos códigos pisen los de otros y cada cierto tiempo (o cuando se termine el proyecto) se combinan todas las ramas se conbinan en la rama principal formando un mecanismo más ordenado que si cada vez que una persona tuviera que poner su código de vuelta en el repositorio, tuviera que descargarse el anterior, modificar su codigo para combinarlos manualmente y próximamente subirlo.
