¿Qué son los boilerplates?

Muchas veces escuchamos o leemos la palabra 'boilerplate' o 'texto repetitivo' en el contexto de programación y no sabemos a qué se refiere, aquí te explicaremos qué es y algunos boilerplates hechos por Orange Cat para poder ayudarte en tus proyectos.

Un boilerplate es una plantilla multiuso para usar en distintas ocasiones, por ejemplo, HTML5 Boilerplate es una página que contiene una plantilla de página web que puedes usar en tus proyectos de desarrollo web.

¿Por qué usar estas plantillas multiuso?

Es recomendable usar boilerplates en vez de ser tú el que escribe el código en el documento por estos motivos:

  • Rapidez: No tienes que tardar tanto en crear el código completo de un archivo si tienes una plantilla que puedes usar.

  • Repetitividad: Un boilerplate siempre es el mismo (a menos que el creador lo cambie) así que no va a poder tener errores cuando escribes lo mismo

  • Buenas prácticas: Un boilerplate siempre va a usar buenas prácticas que puede ser que no conozcas, así que si lo usas, también aprendes nuevas cosas

  • Conexiones y buena estructura principal: Si usas un boilerplate que en vez de un texto sea una carpeta puede que tenga una estructura de archivos más entendible o los archivos accedan a las carpetas con rutas relativas esto es mejor porque accede desde cualquier parte del ordenador donde esté ubicado tu proyecto (incluso si esta en URL)
