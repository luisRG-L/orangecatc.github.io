Orange Cat: El lenguaje de programación

¿Que beneficios aporta Orange Cat?

Estos son los beneficios principales de Orange Cat:

  • Simplicidad: Orange Cat tiene una sintaxis muy simple, con muy pocos tokens puedes crear una página web

  • Rapidez: Está basado en java, así que tiene mucha rapidez

  • Avance: Este lenguaje avanza rápido, ya puedes crear una parte frontend
